Flow #1

Daily/Weekly Portfolio Summary

CRON scheduler tự động fetch data từ Treasury Dashboard, format thành template message, gửi qua GTalk

01
Trigger

CRON Scheduler

Chạy vào 8:00 AM hàng ngày (hoặc 9:00 AM thứ Hai cho weekly report)

0 8 * * * → daily_report() 0 9 * * 1 → weekly_report()
02
Action

Fetch Portfolio Data

Gọi Treasury Dashboard API lấy dữ liệu: summary, loans, investments, limits, insights

GET /api/data → totalLoan, totalInvest, netSpread, netPL...
Total Loan1,204 tỷ
Total Invest1,272 tỷ
Net Spread+1.88%
Net P&L+33.6 tỷ
03
Transform

Build Template Message

Format data thành GTalk Template Message với KPIs, chart summary, và action buttons

📊 Treasury Daily Report — 14/06/2026
💰 Dư nợ: 1,204 tỷ | Đầu tư: 1,272 tỷ
📈 Spread: +1.88% | P&L: +33.6 tỷ
🔴 Room cạn: Cathay, VCB, Shinhan
⚠️ Duration mismatch: 6 cặp
04
API Call

GTalk: send-message

POST template message tới GTalk channel của Treasury team

POST /api/gtalk/send-message { channelId: "treasury_team_ch", content: { template: {...} }, oaToken: "***:***" }
05
Done

Delivered to Team

Team nhận notification trên GTalk app, click "Xem Dashboard" để mở Treasury Dashboard trực tiếp

Flow #2

Chatbot Tra cứu Portfolio

User gửi câu hỏi trên GTalk → Webhook → AI xử lý → Trả lời tự động

01
Trigger

User gửi tin nhắn

User nhắn tin trong GTalk channel: "Cho tôi biết room còn lại của BIDV"

👤
Cho tôi biết room còn lại của BIDV?
02
Webhook

GTalk Webhook → Server

GTalk POST webhook payload tới automation server, bao gồm signature verification

POST https://your-server/webhook/gtalk Header: x-gtalk-event-signature: mac=... { "content": "Cho tôi biết room...", "senderId": "user_123", "channelId": "ch_456", "contentType": 0 }
03
Action

Auto Receipt: SEEN + TYPING

Gửi receipt SEEN + TYPING ngay lập tức để user thấy bot đang xử lý

POST /api/gtalk/send-message-receipt receipts: [ { status: 2, ... }, // SEEN { status: 3, ... } // TYPING ]
Bot đang nhập...
04
AI Process

Parse Intent + Query Data

AI phân tích intent → Truy vấn Treasury API → Compile kết quả

🧠 NLP Parse →
🔍 Query DB →
📝 Format Reply

API Call

GTalk: send-message (Reply)

Gửi kết quả tra cứu về GTalk channel dạng text message (Markdown)

Flow #3

File Report (PDF/Excel) qua GTalk

Generate báo cáo → 3-Step S3 Upload → Send file message qua GTalk

01
Trigger

Trigger Source

Có thể trigger bằng CRON (weekly), chatbot command, hoặc manual request

🕐 CRON (Weekly)
💬 Chatbot CMD
🖱️ Manual
02
Generate

Generate Report File

Fetch data từ Treasury API → Render thành PDF/Excel report

📕 Treasury_Report_2026-06-14.pdf 2.4 MB
📗 Portfolio_Data_2026-06-14.xlsx 856 KB
03
3-Step Upload

GTalk File Upload Flow

Upload file lên S3 thông qua GTalk presigned URL flow

3a
Initiate Upload
POST /api/gtalk/initiate-upload
→ PresignedURL, UploadId
3b
PUT to S3
PUT {PresignedURL}
→ 200 OK
3c
Complete Upload
POST /api/gtalk/complete-upload
→ File Id
04
API Call

GTalk: send-message (File)

Gửi file message với fileId vừa nhận từ complete-upload

POST /api/gtalk/send-message { content: { attachment: { items: [{ file: { fileId: "2009...", fileName: "Treasury_Report.pdf", mimeType: "application/pdf", fileSize: 2516582 } }] } } }
05
Done

File Delivered

Team nhận file trên GTalk, download trực tiếp từ app

Technical

System Architecture

Chi tiết kiến trúc kỹ thuật, API endpoints, và data flow

📡 GTalk API Endpoints Used

Endpoint Method Flow Purpose
/api/gtalk/send-message POST Daily Chatbot File Gửi text, template, hoặc file message
/api/gtalk/send-message-receipt POST Chatbot Gửi SEEN, TYPING receipt
/api/gtalk/config-channel-processing POST Chatbot Cấu hình webhook cho channel
/api/gtalk/initiate-upload POST File Khởi tạo upload session, lấy presigned URL
/api/gtalk/complete-upload POST File Hoàn thành upload, lấy file Id
/api/gtalk/create-server-direct-channel POST Daily Chatbot Tạo channel giữa OA và user
/api/gtalk/modify-message POST Chatbot Sửa/xóa message đã gửi
